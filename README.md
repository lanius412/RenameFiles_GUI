# RenameFiles

### Description
Rename all selected files from "01" regardless of extension

### Support : MacOS

### Running the examples
  ```zsh
    git clone github.com/ry0-suke/RenameFiles.git
    cd RenameFiles
    unzip rename.zip
    ./rename
  ```
  
  Browse... Click
  
  Select Files(Command + => multiple select)
  
  Rename Click
  
  Complete!!
